1. Go to the directory that contains the program
2. Type 'gcc Parser.c -o parser into the terminal.
3. Type './parser -a -l -v' into the terminal.
4. The output is printed to the terminal


The -a -l -v directives can be included or not to provide additional output.
All output is also additionally written to all_output.txt file.
You must also include a file called 'input.txt' in the same directory for input.
